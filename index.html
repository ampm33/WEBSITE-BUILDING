Welcome

PranaPhotos is a visual storytelling studio led by Adri Meyer. This site is a multi-page HTML project built to demonstrate clean structure, semantic layout, and working navigation.

The focus here is organization: clear headings, readable sections, and content grouped in a way that’s easy to expand later with styling and more galleries.

Over time, this site will evolve into a stronger portfolio hub, but for now it delivers a solid semantic skeleton that meets the Release 2 requirements.
